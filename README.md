# Proyecto2-DB
Proyecto 2 de Base de Datos 1. Diseño de un sistema en MariaDB que gestiona productos, categorías, proveedores y stock. El modelo permite registrar compras de clientes atendidas por empleados, vinculando múltiples productos mediante tablas de detalle. Ofreciendo una solución eficiente para CRUD y reportes de ventas mediante consultas SQL.
